# Contributing

Thank you for your interest! This project is part of the [SuperInstance](https://github.com/SuperInstance) ecosystem.

## Quick Start

```bash
git clone https://github.com/SuperInstance/categorical-agents.git
cd categorical-agents
cargo build
cargo test
cargo run --example tutorial
```

## Guidelines

- Follow existing code style (enforced by `cargo fmt` and `cargo clippy`)
- Add tests for new features
- Update documentation when changing behavior
- See [org-level contributing guide](https://github.com/SuperInstance/.github/blob/main/CONTRIBUTING.md) for full details

## Reporting Issues

Use [GitHub Issues](https://github.com/SuperInstance/categorical-agents/issues) for bugs and feature requests.
