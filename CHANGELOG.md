# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-01

### Added
- Capability type: agent capabilities as category objects
- AgentCategory: symmetric monoidal category with tensor products
- Protocol: morphisms between capabilities
- Composition: functor and natural transformation support
- AgentFunctor: mapping between agent architectures
- Tutorial example (`cargo run --example tutorial`)
